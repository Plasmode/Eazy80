A simple Z80 SBC capable of running CP/M.  It has two configurations. 
1.  Glueless SBC with Z80, KIO, RAM and ROM.  Parts are available from eBay seller hth-tech for $7.50.
2.  With addition of Quad-OR gate (74xx32) and replacing 128K RAM with 512K RAM, it can run RomWBW.
